# Computer Vision Data Labeller (ArtisticCV)

Formerly, cv-bootstrap.

## Purpose
Data labeller for a computer vision project.

## Compatibility
I am going to only work on .png and .jpeg files to start. I will add conversions as we progress.

## Dependencies
1. cv2
2. tkinter
3. pillow
4. numpy


## Structure
### //Preprocessing/
